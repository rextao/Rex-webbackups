# 时间处理

## day.js

1. https://github.com/iamkun/dayjs
2. 2kb，与moment有相同的api
3. 支持不可变数据和链式操作，并兼容全部浏览器 

## You-Dont-Need-Momentjs

1. https://github.com/you-dont-need/You-Dont-Need-Momentjs
2. 可用于替换moment.js + ESLint插件的日期文件、原生函数列表
